# Публичный репозиторий проекта «Мини-Склад».

Создана конфигурация с минимальным контуром учёта:

Справочники: Товары, Контрагенты, Сотрудники, **Валюты**,

Документы: Поступление , Реализация, Коммерческое предложение, ВводНачальныхОстатков.

Интерфейс: добавлены объекты в подсистему «Мини-Склад», выведены команды на панель.

Регистры: Остатков товаров, Остатки товаров по партиям,  Продажи по ФИФО, **КурсыВалютКРублю**

СКД-отчет Обороты по товарам, Остатки на дату

Печатная форма документов Поступление, Реализация

**HTTP-сервис для загрузки и выгрузки товаров**,

**Регламентные задания:**

**- Выгрузка остатков товаров из регистра накопления Остатки в xml файл,**

**- Запрос актуальных курсов валют через HTTPЗапрос и запись в регистр сведений КурсыВалют,**

**- Обновление справочника Валюты последними данными из регистра КурсыВалют.**

Созданы роли:

\*\*\* Гость - видит только регистр отпускных цен,

\*\*\* Кладовщик - доступны справочники Товары (Создание, Чтение, Изменение, Удаление), Контрагенты (Чтение)

документы Поступления, Реализация, Ввод начальных остатков - (Создание, Чтение, Изменение, Удаление но не Проведение), плюс ограничена видимость полей Цена и Сумма.

Регистры накоплений - остатки товаров, остатки товаров по партиям,

Отчеты Обороты и Остатки

\*\*\* Менеджер - все возможности в 1С предприятии, нет доступа к Конфигуратору

\*\*\* Администратор - все возможности
